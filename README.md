# Fun experiment with images and point clouds.

Made with Three.js, Polymer, Typescript and Webpack.

```html
<!-- Include script -->
<script type="text/javascript" src="/img-scatter.js"></script>
```

```html
<!-- Use custom element -->
<img-scatter src="my-image.jpg"></img-scatter>
```

### Dev server
```
npm run start
```

### Build
```
npm run build
```
